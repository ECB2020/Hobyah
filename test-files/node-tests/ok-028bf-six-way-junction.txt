Test input file for Hobyah, a 1D tunnel vent program.
July 2024
This file created to test the handling of a six-way junction with different
tunnel orientations.  One six-way junction with six connections:
 forward end, forward end, forward end, forward end, forward end,    back end.
This has positive air pressure at the portal in the tunnel named "first".

begin settings
  version 1
  runtype calc
  frictiontype Fanning
  QA1 2
  QA2 Software testing
  QA3 Files that work and test different capabilities
  aero_step 0.1
  aero_time 300
end settings
