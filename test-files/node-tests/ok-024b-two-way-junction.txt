Test input file for Hobyah, a 1D tunnel vent program.
July 2024
This file created to test the handling of two way junctions with different
tunnel orientations.  Four two-way junctions with 4 types of connection:
    forward end of the first segment, back end of the second (node SW1)
    forward end of the first segment, forward end of the second (node SW2)
    back end of the first segment, back end of the second (node SW3)
    back end of the first segment, forward end of the second (node SW4)
This has negative air pressure at the portal in the first tunnel of each
route.


begin settings
  version 1
  runtype calc
  frictiontype Fanning
  QA1 2
  QA2 Software testing
  QA3 Files that work and test different capabilities
  aero_step 0.1
  aero_time 300
end settings
