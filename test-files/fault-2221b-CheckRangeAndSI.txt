Tests an error message raised when we have a non-integer where we
are expecting an integer.  In this version of the error the number
that fails is set as a constant, so there is an extra entry in the
error message.

    runtype plot
    frictiontype Darcy
end settings

  int_pos  1
  real_neg -1.
  real_pos  1.
end constants

begin testblock

end testblock
