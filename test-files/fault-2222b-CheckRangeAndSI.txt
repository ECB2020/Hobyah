Test input file for Hobyah, a 1D tunnel vent program
December 2020
Tests an error message raised when we have a non-number where we
are expecting a real number.  In this version of the error the number
that fails is set as a constant, so there is an extra entry in the
error message.

begin settings
    version 1
    runtype plot
    frictiontype Darcy
end settings

begin constants
  int_neg -1
  int_pos  1
  real_neg -1.
  real_pos  1.
  dud_constant  blah   # A constant that isn't a number.
end constants

begin testblock
  -   -1
  float   dud_constant   # Should be a real number
end testblock

begin plots
end plots
