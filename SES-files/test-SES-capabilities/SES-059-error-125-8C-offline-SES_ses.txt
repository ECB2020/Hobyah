                                                  SIMULATION OF
                         offline-SES file for triggering error 125 in form 8D, so we can figure out how
                         to process the lines of error message after it.
                         Error 125 occurs when the route ends in one of the segments given in form 8D.
                         The program prints the error message then processes the remaining segments
                         while not using them in the route.  This is the most complex SES error to
                         process, because there is a variable count of lines after the error is
                         raised.
                         This file proceeds to the calculation after writing error 125.
